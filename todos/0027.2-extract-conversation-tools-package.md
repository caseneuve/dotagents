## Context

Move the existing assistant-outline, last-assistant-block, bookmark, and session-notes extensions
into one public package while preserving their current behavior and current integration mechanisms.
Do not introduce a new cross-package event/provider API during extraction.

## Acceptance Criteria

- [ ] Create `github.com/caseneuve/pi-conversation-tools` with local checkout `~/git/pi/pi-conversation-tools`.
- [ ] Move `assistant-outline`, `last-assistant-block`, `bookmark`, and `session-notes` as separate explicit manifest resources.
- [ ] Preserve current commands, shortcuts, session entries, branch behavior, editor export, clipboard behavior, and existing event/status integration exactly where practical.
- [ ] Preserve current platform assumptions and document them without expanding support.
- [ ] Move existing tests and make them pass with documented commands.
- [ ] Local-path and immutable-ref Git installs work with supported `@earendil-works/*` APIs.
- [ ] Any generalized cross-package provider API belongs to follow-up `0048`, not this extraction.

## Affected Files

- `pi/extensions/assistant-outline/*`
- `pi/extensions/last-assistant-block.ts`
- `pi/extensions/bookmark.ts`
- `pi/extensions/session-notes.ts`
- New `~/git/pi/pi-conversation-tools` repository.
