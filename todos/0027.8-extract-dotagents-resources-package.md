## Context

Dotagents should remain the personal agent orchestration, shared-skills, and bootstrap repository,
not the canonical implementation home for distributed Pi resources. Its current Pi themes and
prompt templates need a resource-only package that can be installed locally or from Git.

## Acceptance Criteria

- [ ] Create `~/git/pi/pi-dotagents-resources` using the shared package convention.
- [ ] Package the current themes and prompt templates with explicit manifest entries.
- [ ] Local-path and pinned Git installs expose the same theme names and prompt commands.
- [ ] The `modus-operandi` default continues to work after cutover.
- [ ] README documents global installation and resource filtering.

## Affected Files

- `pi/themes/*` — resources to migrate.
- `pi/prompts/*` — resources to migrate.
- New `~/git/pi/pi-dotagents-resources` repository — canonical implementation.

## Notes

This is a resource-only package; do not use it as a new home for executable extensions.
